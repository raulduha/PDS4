# 📦 Proyecto pds4 - Estación de Casilleros

Bienvenido al proyecto pds4. Estamos creando una innovadora Estación de Casilleros que permite a los usuarios reservar, cargar, retirar paquetes y más.

---

## 🛠 Tecnologías utilizadas

- **Backend**: Django
- **Frontend**: React.js

---

## 🚀 Inicialización

### Backend (Django)

#### 📋 Requisitos:

- Python 3.9.7
- pip 23.2.1

#### 🚀 Instrucciones de inicialización:

1. **Navega a la carpeta `backend`:**
    ```bash
    cd backend
    ```

2. **Activa el entorno virtual:**
    ```bash
    source venv/bin/activate  # En sistemas basados en UNIX
    venv\Scripts\activate  # En Windows
    ```

3. **Instala las dependencias:**
    ```bash
    pip install -r requirements.txt
    ```

4. **Ejecutar el servidor:**
    ```bash
    python manage.py runserver
    ```

5. **Creación de superusuario:**
    ```bash
    python manage.py createsuperuser
    ```

---

### Frontend (React.js)

#### 📋 Requisitos:

- Node.js
- npm o yarn

#### 🚀 Instrucciones de inicialización:

1. **Navega a la carpeta `frontend`:**
    ```bash
    cd frontend
    ```

2. **Instala las dependencias:**
    ```bash
    npm install
    # o
    yarn install
    ```

3. **Ejecutar la aplicación:**
    ```bash
    npm start
    # o
    yarn start
    ```
